## Documentation

### Installation

```shell
$ git clone {repo url}
$ cd {reponame}
$ pip install -r requirements.txt

### How To Start

```shell
$ python main.py
```

### Dependency

- Python==3.10
- django==3.0.0
- requests^0.10.0

### Features

- 행운의 숫자 추출

- 행운의 숫자 기반 구매



## vi로 작업하는 법

```text
ex) .gitignore

1. `$ vi .gitignore`

2. vim에서
	
	a. normal mode -> insert mode: `i`
	b. 작업 실시 (cmd(ctrl)+v 또는 mousw(right)->paste)
	c. 작업 끝났으면 insert mode -> normal mode: `esc`
	d. normal mode -> command mode: `:`
	e. Write and Quit: `:wq` + enter
```
